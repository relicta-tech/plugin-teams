## [2.0.0] - 2024-12-17

### Added
- Initial release as standalone plugin using the Relicta Plugin SDK
- Full gRPC-based plugin communication
- Comprehensive configuration validation
- Dry run support for safe testing
- Environment variable fallback for credentials

### Changed
- Migrated from embedded plugin to standalone SDK-based architecture
- Improved error handling and reporting
